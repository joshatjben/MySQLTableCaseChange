MySQL Table Case Changer

Convert the case of MySQL table names from a mysqldump file. Helpful for the pesky windows to linux issue. This is the online version.

What is it? cose

Did you ever have a problem when deploying a MySQL DB from a Windows machine to a Linux machine where your table names are lowercase on windows, but uppercase under Linux? And maybe to make matters worse you have no control over the shared hosting Linux environment to affect the table name case sensitivity!? Well this application can hopefully help you out. It can take an input from say, a mysqldump output, and change all the table names to either all upper or all lower case names.
