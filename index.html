MySQL Table Case Changer

Convert the case of MySQL table names from a mysqldump file. Helpful for the pesky windows to linux issue. This is the online version.

What is it? hide

Ever have trouble with MySQL table names while moving between Windows and Linux environments. Specifically involving case (in)sensitivity? And to make matters worse your running on a shared hosting environment that you have no contorl over? This quick dirty app can take text from say, a mysqldump output, and changes all the table names to all upper or lower case names. Way easy.

Disclaimer: A bulk change to all your table names is a pretty crazy thing to do, back up back up back up!

Here's how to convert your entire db's to UpperCase table names.
  1. Open a console and run mysqldump -h[host] -u[userName] -p[password] --add-drop-table --add-drop-database [dbname].
